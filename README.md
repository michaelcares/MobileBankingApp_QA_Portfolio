# MobileBankingApp_Manual_QA_Portfolio
Manual QA testing project simulating a full test cycle
on a mobile banking application.

# What's inside
- Test Plan (scope, tools, entry/exit criteria)
- 30 test cases across 6 modules
-  5 detailed JIRA bug reports

# Tools used
JIRA · Google Sheets · Android Emulator

# Modules tested
Registration · Login · Fund Transfer ·
Airtime Purchase · Transaction History · Settings
